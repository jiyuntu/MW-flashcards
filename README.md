# MW-flashcards
The program takes user input, connects to Merriam-Webster API, and makes an Anki flashcard. For more information, look up https://jiyuntu.github.io/blog.
## Requirements
1. [Anki Desktop](https://docs.ankiweb.net/platform/windows/installing.html#installing): Keep Anki Desktop open while using the program.
2. 
```
pip install -r requirements.txt
```

## Usage
```
python main.py
```

Right after the cards are added, you will be able to view them from Anki Desktop. To view it from Anki App, sync it to Anki Desktop.
